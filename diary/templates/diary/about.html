  • Obsolete Jobs

    Obsolete jobs (such with a result assessment set to "obsolete") are hidden from the job list by default. It is still possible to access the detail page of that job via it's job id in the URL (e.g. /1234567/). To show obsolete jobs in the job list, add /?show_obsolete to the URL like this. If there are other query parameters present in the URL, the show_obsolete option can be chained via & like this.
